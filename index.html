Responsive Scaling

Media queries

Media queries

  • min-width queries
  • max-width queries

min-width media queries

// Note: Always write media queries in `em`.
.component {
  @media (min-width: 37.5em) {
    font-size: 2em;
  }
}

max-width media queries

.component {
  font-size: 2em;
  @media (max-width: 37.5em) {
    font-size: 1em;
  }
}
