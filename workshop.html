Responsive Scaling

Media queries

  • min-width queries
  • max-width queries

Discuss (5mins)

  • When should you use min-width queries?
  • When should you use max-width queries?

min-width media queries

// Note: Always write media queries in `em`.
.component {
  @media (min-width: 37.5em) {
    font-size: 2em;
  }
}

max-width media queries

.component {
  font-size: 2em;
  @media (max-width: 37.5em) {
    font-size: 1em;
  }
}

Quick tip for responsive scaling

  • Keep your media queries near your component
.component {
	/* style */
}

@media screen and (min-width: 37.5em) {
	.component {
	  /* style */
  }
}

Responsive Scaling:

  • media queries
  • favor min-width over max-width
  • use both when you need to!
